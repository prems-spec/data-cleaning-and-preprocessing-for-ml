# Data Cleaning and Preprocessing for Machine Learning

## 📌 Project Overview

This project demonstrates essential **data cleaning** and **data preprocessing** techniques used before training machine learning models.

The repository includes examples of detecting, visualizing, and handling outliers using statistical methods and machine learning techniques.

---

## 🚀 Features

* Compare the effect of outliers on the mean
* Detect outliers using **Z-Score**
* Detect outliers using **Interquartile Range (IQR)**
* Visualize outliers with **Box Plot**
* Visualize outliers with **Scatter Plot**
* Handle outliers using **Winsorization**
* Detect anomalies using **Isolation Forest**
